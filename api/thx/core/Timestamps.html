class thx.core.Timestamps

import thx.core.Dates;

Available on all platforms

Timestamps provides additional extension methods on top of the Float type representing date values.

using Dates;

@author Jason O'Neil @author Franco Ponticelli

Class Fields

static inline function create(year:Int, ?month:Int, ?day:Int, ?hour:Int, ?minute:Int, ?second:Int):Float

Creates a date (timestamp/float format) by using the passed year, month, day, hour, minute, second.

Note that each argument can overflow its normal boundaries (e.g. a month value of -33 is perfectly valid) and the method will normalize that value by offsetting the other arguments by the right amount.

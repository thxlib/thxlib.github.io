abstract thx.Set<T>(Array<T>)

Available on all platforms

A set is a list of unique values.

Class Fields

function arrayToSet<T>(arr:Array<T>):Set<T>

arrayToSet converts an Array into a Set removing all duplicated values.

function create<T>(?arr:Array<T>):Set<T>

Creates an empty Set if no argument is provided or it fallsback to arrayToSet otherwise.

Instance Fields

function add(v:T):Bool

add pushes a value onto the end of the Set if the value was not already present.

It returns a boolean value indicating if the Set was changed by operation or not.

inline function copy():Set<T>

copy creates a new Set with copied elements.

inline function difference(set:Set<T>):Set<T>

difference creates a new Set with elements from the first set excluding the elements from the second.

function exists(v:T):Bool

exists returns true if it contains an element that is equals to v.

inline function get(index:Int):Null<T>

get returns the element at the specified position or null if the index is outside the boundaries.

inline function intersection(set:Set<T>):Set<T>

intersection returns a Set with elements that are presents in both sets

function push(v:T):Void

Like add but doesn't notify if the addition was successful or not.

function setToArray():Array<T>

Converts a Set<T> into Array<T>. The returned array is a copy of the internal array used by Set. This ensures that the set is not affected by unsafe operations that might happen on the returned array.

inline function slice(pos:Int, ?end:Int):Set<T>

Same operations as Array.slice() but it returns a new Set instead of an array.

inline function splice(pos:Int, len:Int):Set<T>

Same operations as Array.splice() but it returns a new Set instead of an array.

function toString():String

Converts Set into String. To differentiate from normal Arrays the output string uses curly braces {} instead of square brackets [].

inline function union(set:Set<T>):Set<T>

Union creates a new Set with elements from bots sets.
